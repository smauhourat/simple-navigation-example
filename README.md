# Simple Navigation Example

A React Native application demonstrating basic navigation patterns.

## Getting Started

### Prerequisites
- Node.js and npm
- React Native CLI
- Android Studio or Xcode (for native development)

### Installation

```bash
npm install
```

### Running the App

**iOS:**
```bash
npx react-native run-ios
```

**Android:**
```bash
npx react-native run-android
```
